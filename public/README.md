Place your resume PDF in this `public` folder so it can be served at the project root.

- Save your file as: `AnanAlmasri.pdf`
- After adding, the file will be available at: `http://localhost:5175/AnanAlmasri.pdf`

Example:
- public/AnanAlmasri.pdf -> /AnanAlmasri.pdf

Notes:
- Vite serves files placed in `public/` at the project root during development and in the built app.
- If you change the filename, update the `Hero` download link in `src/components/Hero.tsx` to point to the new name.
